# Platform Purpose

## Summary

Build a scalable digital platform for the equine industry with an initial focus on racehorses.

## Primary Goals

- capture structured performance data
- capture structured biochemistry data
- provide personalised member access
- support multiple membership levels such as owners and trainers
- support e-commerce
- support a database-backed website and app ecosystem

## Recommended Stack

- Antigravity for orchestration
- GitHub for version control
- Supabase for database, auth, and storage
- Vercel for web deployment
- Railway for background services and integrations
- GoDaddy as the domain registrar

## Notes

- The platform should be designed to scale beyond a marketing website.
- The access model must support future role expansion.
- Data architecture will be central to product value.
